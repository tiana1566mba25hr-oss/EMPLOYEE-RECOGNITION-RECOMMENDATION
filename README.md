# AI Recommendation Workflow Automation Dashboard

## 📌 Overview
This project showcases an AI-powered workflow integrated with Google Sheets and Power BI to automate employee performance analysis. It classifies employees into **Highly Recommended**, **Recommended**, and **Not Recommended** categories for better decision-making.

---

## 🚀 Features
- Automated workflow using data pipeline
- Recommendation classification column
- Real-time Google Sheets integration
- Interactive Power BI dashboard
- Department-wise performance insights
- Employee-level detailed analysis

## 📁 Dataset
The dataset includes:
- Employee Name
- Department
- Performance Score
- Attendance
- Feedback Rating
- Recommendation Category

---

## ⚙️ Workflow Explanation
1. Data is fetched from Google Sheets  
2. Workflow processes employee records  
3. Recommendation logic categorizes employees into:
   - Highly Recommended  
   - Recommended  
   - Not Recommended  
4. Updated data is pushed back to the sheet  
5. Power BI dashboard visualizes insights
